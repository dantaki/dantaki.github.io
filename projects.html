Fun Projects

DeepBake

Baking Machine Learning into the Great British Bake Off.

DeepBake is a series of convolutional neural networks, one for each episode. Each model predicts the final placing for each baker. Incidentally, the worst performing remaining baker is typically the one eliminated. Spooky. Learn more here

vapeplot

matplotlib extension for vaporwave aesthetics.

Vapeplot is a simple package to help users color matplotlib plots according to 90s color schemes. Apparently, a very popular aesthetic with nearly 500 stars on GitHub. GitHub Repo

Marcus Aurelius Twitter Bot

Tweeting wisdom to world leaders.

Not a true automated bot, but a command line tool to help you tweet excerpts from Marcus Aurelius' Meditations. The user selects a twitter handle and the number of recent tweets to mine. The script will match words from the tweets to those in Meditations. Then the user will pick the excerpts they like. Below is an example of use, tweeting at Boris Johnson since he has a degree in Classics. GitHub Repo

pylemma

Draw an analemma anytime anywhere (on Earth)

There are probably less than 200 or so photographs of an analemma ever made. Before digital photography this number was less than 50. Provide the latitude, longitude, and time of day and pylemma will draw out the annual path of the Sun. GitHub Repo

fretter

Fret your notes in matplotlib

Simple jupyter notebook containing a Music object with attributes for certain scales and chords. The Guitar object uses matplotlib to visualize the notes on a guitar. GitHub Repo
